# ChatList_1

ChatList — настольное приложение на Python с графическим интерфейсом PyQt6. Приложение используется для отправки одного запроса нескольким нейросетям и для хранения ответов и промтов.

PROJECT.md – описание проекта


## Минимальная программа PyQt

Простое окно с надписью и кнопкой на PyQt6.

### Требования

- Python 3.10 или новее
- Windows PowerShell

### Установка зависимостей

```powershell
cd C:\Work\ChatList_1
python -m pip install -r requirements.txt
```

### Запуск программы

```powershell
cd C:\Work\ChatList_1
python main.py
```

После запуска откроется окно с надписью «Привет, PyQt!». Нажмите кнопку «Нажми меня», чтобы изменить текст на «Минимальная программа на Python».
