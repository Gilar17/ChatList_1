ChatList

ChatList — это приложение на Python для отправки одного промпта в несколько нейросетей через OpenRouter и сравнения полученных ответов.

Основные возможности

  • Отправка промпта нескольким моделям одновременно
  • Сравнение и сохранение результатов
  • Управление промтами и моделями
  • AI-ассистент для улучшения промптов
  • Создание альтернативных и адаптированных вариантов промпта
  • Экспорт результатов в Markdown и JSON
  • Настройка темы и размера шрифта

Используемые технологии

  • Python
  • PyQt6
  • SQLite
  • OpenRouter API

Системные требования

  • Windows 10 или Windows 11
  • Админ-права не требуются (можно установить в пользовательскую директорию)

Установка

  1. Перейдите на страницу релиза: последний GitHub Release.
  2. Скачайте файл ChatList-1.0.1-Setup.exe.
  3. Запустите установщик и следуйте инструкциям мастера.

Важное про OpenRouter API-ключ

Для работы AI-ассистента и генерации ответов ChatList использует OpenRouter API. Нужен собственный API-ключ.

Пожалуйста, не публикуйте ваш OPENROUTER_API_KEY в GitHub.

В установленной версии ключ сохраняется через Настройки → OpenRouter API в Windows Credential Manager. При разработке из исходников можно также использовать файл .env.example (не публикуйте реальный ключ).
